# ElevenLabs Style Reference

Architect's blueprint on warm vellum: light serif headlines at 48px with `-0.02em` tracking, anchored against an eggshell ground, with pure black pill buttons as the main punctuation.

## Theme

Light-first, warm, type-led, near-monochrome.

## Core Tokens

- Eggshell: `#fdfcfc`
- Powder: `#f5f3f1`
- Chalk: `#e5e5e5`
- Fog: `#b1b0b0`
- Gravel: `#777169`
- Slate: `#a59f97`
- Cinder: `#57534f`
- Obsidian: `#000000`
- Signal Blue: `#0447ff`
- Ember: `#ff4704`

## Typography

- Display: Waldenburg-style serif, fallback `Lora`, `Cormorant Garamond`, or `Libre Baskerville`.
- Body: Inter-style sans, fallback system UI.
- Mono: Geist Mono-style, fallback `Fira Code` or `JetBrains Mono`.
- Display size: `48px`, line-height `1.08`, tracking `-0.02em`.
- Heading large: `36px`, line-height `1.13`, tracking `-0.02em`.
- Body: `14-16px`, line-height `1.43-1.5`.

## Layout And Components

- Page background: Eggshell, no decorative grid.
- Borders: Chalk.
- Secondary text: Gravel.
- Buttons and tags: pill radius.
- Cards: `16px` radius.
- Panels: `20px` radius.
- Surfaces use subtle inset hairline shadows instead of heavy elevation.
- Use major vertical gaps of `80-120px`; use `8-12px` for local component spacing.

## Rules

- Keep the palette almost achromatic.
- Do not use saturated colors for text, backgrounds, or major buttons.
- Reserve Signal Blue and Ember for tiny dot/avatar indicators only.
- Keep headings light and restrained.
- Prefer warmth, quiet spacing, and typography over decorative graphics.
