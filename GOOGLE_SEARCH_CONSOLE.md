# Google Search Console Setup Guide

## Step 1: Verify Ownership

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add property" and select "Domain" or "URL prefix"
3. If using URL prefix, enter: `https://suhasdarsi.com`
4. Choose a verification method:

### Option A: HTML File Upload (Recommended)
- Download the verification HTML file from Google
- Place it in the `public/` folder of this repo
- Deploy the site
- Click "Verify" in Search Console

### Option B: DNS TXT Record
- Add the TXT record provided by Google to your DNS settings (Cloudflare)
- Wait a few minutes for propagation
- Click "Verify"

### Option C: Meta Tag
- Copy the meta tag provided by Google
- Add it to `src/layouts/BlogLayout.astro` in the `<head>` section:
```astro
<meta name="google-site-verification" content="YOUR_CODE_HERE" />
```
- Deploy and verify

## Step 2: Submit Sitemap

1. In Search Console, go to "Sitemaps" in the left sidebar
2. Enter: `sitemap-index.xml`
3. Click "Submit"

The sitemap is automatically generated by `@astrojs/sitemap` at build time.

## Step 3: Check Indexing Status

1. Go to "Pages" in the left sidebar
2. Review which pages are indexed vs not indexed
3. Request indexing for important pages if needed

## Step 4: Monitor Performance

- Check "Performance" tab for search queries and clicks
- Review "Core Web Vitals" for page speed metrics
- Fix any "Mobile Usability" issues
